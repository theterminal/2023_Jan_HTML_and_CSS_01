Box-sizing

The default state of the property box-sizing is 'box-sizing: content-box;' which means that the width of the container that we define will be only for the content and NOT including padding and border. Margins are ALWAYS outside of the box.

The other way, to have everything calculated in the with you define is 'box-sizing: border-box;'. In this case we define the maximum width of the container from the border inword, including the border.

Good practice - CSS Box-sizing Reset

It is a good practice to do the following at the top of the CSS document:

html {
box-sizing: border-box;
}

*,
*:before,
*:after {
box-sizing: inherit;
}
