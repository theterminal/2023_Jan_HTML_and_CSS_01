Float is used mostly to wrap text arround image. Used to be different but now with all the modern tools availabe that is about the only use of 'float'.

Do not use 'float' to create layouts! The 'float' property creates a new document flow. Take a look at the exmple. If you uncomment the 'clearfix' class in the CSS, the image will get inside the border of the container containing it. That is an old way to use 'float' property to do disign layout.

Using 'float' to do a page layout is a hack not a normal workflow.
