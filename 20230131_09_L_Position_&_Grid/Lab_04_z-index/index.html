All elements have initial z-index: 0. If two or more elements have same index value, they will appear on the screen based on their order in the HTML document (the later in the HTML the higher priority).

The higher the z-index the closer to you is the element. z-index is the 'z' axis of the coordinate system with '0 position' at the screen level. There are negative indexes also...

It is possible to use 'z-index' as a group but the container of the group must NOT have 'position: static'. It must be positioned with some other value than 'static'.

The elements inside a container which container has z-index set, have z-index of the container comparing the z-indexes of all elements outside of the container but inside the container the elements are compairing to one another with their z-indexes. The actual z-indexes inside the container matter between the elements inside but as far as outside the container the group has the z-index of the container...

See the example above. The elements 4, 5, 6 are in a group with z-index: 20 of the group.
