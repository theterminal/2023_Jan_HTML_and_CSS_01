2. Font size px, em, pt

In the following 'ul' element in the 'class' 'main-ul' the font-size is set to 18px.

.main-ul {
font-size: 18px;
}

There is no elements directly affected by the 18px declaration.

Then the font-size is set to '1.5em' which is 1.5 * 18px = 27px

The target are all 'li' elements in an 'ul' elments

.main-ul li {
font-family: "ShadowsIntoLight-Regular";
font-size: 1.5em;
}

The elements directly affected by the declaration are the direct children 'li' of the 'ul' element with class '.main-ul'

Then the font-size is set to '1.5em' which is 1.5 * 27px = 40.5px

The target elements are all 'li' elements in an 'ul' element which 'ul' element is in 'li' element which 'li' element is in element with class '.main-ul'

.main-ul > li > ul > li {
font-family: "ShadowsIntoLight-Regular";
font-size: 1.5em; }

If the nesting of the elements continues, the size is getting larger by the same factor 1.5

3. Font size px, rem

When setting 'rem' value of the 'font-size' property, the size of the font is calculated based on the font size set at 'html' level.

As it seen bellow the 'Level 1' 'li' elements (set to '2rem') are with the size of 36px, because the 'html' font size is set to 18px.

The size of 'Level 2', 'Level 3', 'Level 4' elements (level 2 and above are set to 1.5em) is 54px (36px * 1.5em)
