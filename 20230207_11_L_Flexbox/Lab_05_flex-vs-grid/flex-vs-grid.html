Explanation of Grid and Flex and the main diffetence

  • When we use 'GRID' we create boxes, we create columns and rows with defined sizes, where our elemets live.

  • 'FLEX' doesn't control the size of its children. Every element fits next to one another and we can control how the elements fit within the container.

  • 'FLEX' keeps the size of its children where the 'GRID' defines exact dimentions and the children fit inside. This is the main difference!

  • If for us is important the size of the children, we use 'FLEX'. If we want elements with same dimentions we use 'GRID'. With grid we define space that we fill where with 'GRID' we fill space using elements...

  • FLEXBOX layout is most appropriate to the components of an application, and small-scale layouts, while the GRID layout is intended for larger scale layouts.
