Pseudo Elements have 2 colons (::) and Pseudo Classes have only one colon (:)
Examples: .card::before {property: value;} - this is Pseudo Elemet (it does NOT exist in the HTML but it is visible only)
.card:hover {property: value} - this is a Pseudo Class (it exists in the HTML)
