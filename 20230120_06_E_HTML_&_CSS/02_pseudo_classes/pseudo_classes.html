'span' element

The element 'span' is inline element similar to 'p' element. For example if you want to style a sentence out of a paragraph differently, 'span' can be used. It is a container for text as 'p' element. 'p' element is a blck type element where 'span' is a inline element.

Example

I will use 'span' element to make the next sentence darkgreen color, upper case, font-weight 700, and the rest will continue to be black text. Quis recusandae laborum dolorum itaque blanditiis. That is how you do it! Of course classes, ids and everything else can be used if needed...

CSS '@import' property

@import is used in the CSS file when we need to import another CSS file (local or from the internet)

Examples

The linking is done same way whether or not it is a local or internet linked file.This is how the importing of font awesome icons is done at the CSS file:

            
    @import url("https://use.fontawesome.com/releases/v5.10.2/css/all.css");

The following CSS import is for importing the google font 'Roboto'. It is done by the following declaration in the CSS file.

@import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap');

CSS '@font-face' property

'@font-face' property is used when a font family needs to be declared because it is unknown to the browser

Examples

The following is an example of @font-face property and it tells the browser what is the font-family name, font-style, font-weight, and the source file where the font details actually are...

            
    /* cyrillic-ext */
    @font-face {
    font-family: 'Roboto';
    font-style: italic;
    font-weight: 100;
    src: url(https://fonts.gstatic.com/s/roboto/v30/KFOiCnqEu92Fr1Mu51QrEz0dL-vwnYh2eg.woff2) format('woff2');
    unicode-range: U+0460-052F, U+1C80-1C88, U+20B4, U+2DE0-2DFF, U+A640-A69F, U+FE2E-FE2F;
    }
    /* cyrillic */
    @font-face {
    font-family: 'Roboto';
    font-style: italic;
    font-weight: 100;
    src: url(https://fonts.gstatic.com/s/roboto/v30/KFOiCnqEu92Fr1Mu51QrEzQdL-vwnYh2eg.woff2) format('woff2');
    unicode-range: U+0301, U+0400-045F, U+0490-0491, U+04B0-04B1, U+2116;
    }
    /* greek-ext */
    @font-face {
    font-family: 'Roboto';
    font-style: italic;
    font-weight: 100;
    src: url(https://fonts.gstatic.com/s/roboto/v30/KFOiCnqEu92Fr1Mu51QrEzwdL-vwnYh2eg.woff2) format('woff2');
    unicode-range: U+1F00-1FFF;
    }
